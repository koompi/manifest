# PIONUX-MOBILE OS Manifest

## Getting Started

To get started with Android/PIONUX OS, you'll need to get familiar with
[Repo](https://source.android.com/source/using-repo.html) and [Version Control
with Git](https://source.android.com/source/version-control.html).

To initialize your local repository using the PIONUX OS trees, use a command like this:

    repo init -u git://github.com/pionuxos/manifest.git -b pionuxos-android-8.1

Then to sync up:

    repo sync
